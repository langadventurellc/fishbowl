# Documentation

This directory contains all project documentation organized by category.

## Structure

- **specifications/** - Technical specifications and architecture documents
- **guides/** - User guides, tutorials, and how-to documentation
- **technical/** - Technical documentation, API references, and developer resources
- **blackboard.md** - Agent collaboration knowledge base

## Documentation Standards

- Use Markdown format for all documentation
- Include cross-references between related documents
- Keep documentation up-to-date with code changes
- Use descriptive filenames with date prefixes where appropriate
